# 🏈 Momentos NFL

Proyecto fullstack desarrollado con:

- Angular
- Node.js
- Express
- MongoDB
- Autenticación JWT
- Roles (Admin / User)
- MFA por código
- Docker (opcional)

## 📌 Funcionalidades

- Carrusel dinámico
- CRUD de historias
- Historias sensibles con disclaimer
- Panel admin
- Control de roles
- Animaciones AOS

## 🚀 Instalación

### Backend
cd BACKEND
npm install
npm start

### Frontend
cd PROYECTO-FINAL-ANGULAR
npm install
ng serve
